How to install:

1. Copy all files of the extension (phpdocx.php and phpdocx.xml) to the plugins folder in a new directory:

plugins/content/phpdocx

2. Copy all files of the library package to a new phpdocx directory in the Joomla libraries folder (plugins/phpdocx).

3. Open the Joomla admin interface and go to Extensions->Manage->Discover section.

4. Select the phpdocx extension and install it.

5. Go to Extensions->Manage->Manage and enable the phpdocx extension.

6. Change the file phpdocx.php to use the library with the preferred events (onContentAfterTitle as default).
